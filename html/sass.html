SASS Syntax

  • basic sass
#header, #footer
  :background-color #333
  :margin 5px 10px 3px 1px
#header, #footer {
  background-color: #333;
  margin: 5px 10px 3px 1px;
}
  • nesting
#header
  :background-color #333
  ul
    :list-style-type none
    a
      :text-decoration none
#header {
  background-color: #333333;
}
#header ul {
  list-style-type: none;
}
#header ul a {
  text-decoration: none;
}

    • loud comments convert to comments in the css output, and are marked
      with /*
